# To-Do List App V2

A menu-driven To-Do List application built with Python.

## Features

* Add Tasks
* Remove Tasks
* View Tasks
* Search Tasks
* Clear All Tasks
* Total Tasks Counter
* Input Validation
* Duplicate Prevention
* Save Tasks to File
* Load Saved Tasks Automatically

## Concepts Used

* Lists
* Loops
* Conditions
* While Loop
* User Input
* File Handling
* Exception Handling (try-except)

## How It Works

Tasks are automatically saved to a text file when the program exits. When the application starts again, previously saved tasks are loaded automatically.
